Python函数注释

Python 函数注释是在PEP 3107引入的，其主要目的是给函数形参和返回值添加可选元数据。

两点重要的信息

  1. 函数注释可选
  2. 可添加任意 Python 表达式，但是 Python 本身并不会对表达式做处理，也不具有任何意思

函数注释可以被用来实现很多功能：

  • 类型检测
  • 谓词逻辑
  • IDE 类型辅助

我们先创建一个函数，并给两个参数添加_谓词注释_。

  1. times 必须大于 0 并且小于 10
